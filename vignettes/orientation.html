Getting Started with betydata

Offline Access to BETYdb Plant Trait Data

Akash B V

2026-02-11

Introduction

The betydata package provides offline access to public data from BETYdb, the Biofuel Ecophysiological Traits and Yields database. This vignette demonstrates the basics of loading and exploring the packaged datasets.

Key features:

  • No database connection required
  • Versioned, reproducible data snapshots
  • Pre-joined, analysis-ready data frames
  • Compatible with tidyverse workflows

Loading the Package

library(betydata)

Column Description Type
trait Variable name (e.g. “SLA”, “Vcmax”, “Ayield”) character
mean Observed value numeric
units Measurement units character
scientificname Species scientific name character
sitename Research site name character
author Citation author character
checked QA/QC status (0 = unchecked, 1 = verified) integer

Example: Bioenergy Crop Yields

library(dplyr)

# Define bioenergy genera
bioenergy_genera <- c("Miscanthus", "Panicum", "Populus", "Salix", "Saccharum")

# Extract yield data
yields <- traitsview |>
  filter(
    trait == "Ayield",
    genus %in% bioenergy_genera,
    !is.na(mean)
  ) |>
  select(genus, mean, units, sitename, author, citation_year, lat, lon)

# Summary by genus
yields |>
  group_by(genus) |>
  summarise(
    n = n(),
    mean_yield = mean(mean, na.rm = TRUE),
    sd_yield = sd(mean, na.rm = TRUE),
    .groups = "drop"
  )
#> # A tibble: 5 × 4
#>   genus          n mean_yield sd_yield
#>   <chr>      <int>      <dbl>    <dbl>
#> 1 Miscanthus  1021       12.8    10.5 
#> 2 Panicum     2087       10.1     5.46
#> 3 Populus      841       29.3    29.0 
#> 4 Saccharum   3578       34.9    22.1 
#> 5 Salix        528       15.8    18.3

Working with Plant Functional Types (PFTs)

data(pfts)
data(pfts_species)

# Find PFTs for common bioenergy species
data(species)
miscanthus_sp <- species |> 
  filter(genus == "Miscanthus") |>
  pull(id)

# Which PFTs include Miscanthus species?
pfts_species |>
  filter(specie_id %in% miscanthus_sp) |>
  left_join(pfts |> select(id, name), by = c("pft_id" = "id")) |>
  distinct(name)
